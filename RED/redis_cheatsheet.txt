HSET is a command used with Redis Hashes, not Sets.
## IS UNORDERED
A Redis Hash is essentially a map or dictionary:
key -> field -> value
 Store object-like data : { key1:1value, key2:value2...}
 ğŸŸ¥ HGETALL h_test             -> return all keys and values
 ğŸŸ¥ HVALS h_test               -> return all vals
 ğŸŸ¥ HGET h_test c1             -> return h_test[c1] (value)
 ğŸŸ¥ HSET h_test field1 "Hello" -> add {field1:"Hello"}
 ğŸŸ¥ HINCRBY h_test joon 1      -> add 1 HSET (one str at a time)
 ğŸŸ¥ HMGET h_test field1 field2 -> get multiple field values
